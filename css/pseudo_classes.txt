used to control state of selectors

Eg
selector:after{} #end of selector
selector:hover{} #activate css on hover over the selector

tag:action{}
Eg: a:hover{}

first and last child selectors:
selector tag:first-child
selector tag:last-child

first and last type selectors:
selector tag:first-of-type
selector tag:last-of-type

nth child selectors:
selector tag:nth-child(number)
li:nth-child(1)
li:nth-child(even)
li:nth-child(2n+1) #n=0

nth type selectors:
selector tag:nth-of-type(number)
li:nth-of-type(1)
li:nth-of-type(even)
li:nth-of-type(2n+1) #n=0of-type
