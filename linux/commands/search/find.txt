command: find <dirname> <options> <type>  <options> <type>
find particular files in a directory

options:
-type: type of files to get
-name: name of files needed to be found
-iname: case insensitive find
-mmin: files modified minutes ago
-mtime: files modified days ago
-amin: files accessed minutes ago
-atime: files accessed days ago
-cmin: files changed minutes ago
-ctime: files changed days ago
-size: files havig size
-empty: empty files
-perm: files having certain permissions
-exec: execute command
-maxdepth : how many child directories to scan


type:
f - list all files
d - list all dirs
 
Eg:
find . -type f -name "test*" get all files starting with test
find . -type f -mmin -10 list all files modified 10 mins ago
find . -size +5M list all files greater then 5MB
find . -size +5M -exec chmod 775 {} + list all files greater then 5MB amd change file permission
find . -type f -name "*.jpg" -maxdepth 1 list all files with ending .jpg in current directory

note: all -exec commands end with {} +
