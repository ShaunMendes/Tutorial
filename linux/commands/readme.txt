Terminal Shortcuts:
Ctrl+a beginning of line
Ctrl+e end of line
Ctrl+u delete everything before cursor
Ctrl+k delete everything after cursor
!fi runs the last command starting with fi
!<commnd_number> runs the command corresponding to no in history
Ctrl+r to search for commands in history
clear to clear terminal

alias to show all alias
